Posted on 2020-01-05

markdown 基本语法

标题

  1. # + 你的标题 （一级标题）
  2. ## + 你的标题 （二级标题）
  3. ### + 你的标题 （三级标题）
  4. #### + 你的标题 （四级标题）
  5. ##### + 你的标题 （五级标题）
  6. ###### + 你的标题 （六级标题）

    内容

    1. 缩进

  • 手动输入空格：& nbsp; //注意&后没有空格，”;”为半角输入——如英文输入法下的分号
  • 全角状态下——如中文输入法，直接键入空格键

  1. 换行

  • 两次回车（enter键）
  • 使用< br> ，注意：实际使用时“<>”内没有空格

  1. 字体

  • 大小：< font size=5>size
  • 颜色：< font color=red>color
  • 类型：< font face=”微软雅黑”>微软雅黑
  • 加粗：**内容示例**
  • 斜体：*内容示例*
  • 斜体加粗：***内容示例***
  • 分割线：在某一空行键入 — 或者 ***

  1. 图片

  1. 超链接

  • [超链接名](超链接地址 “超链接title”)
    title可加可不加
    示例：
    简书
    百度
  • 或者使用HTML的a标签代替
    <a href=”超链接地址” target=”_blank”>超链接名</a>
    示例：
    简书

  1. 列表

  • 无序列表
    语法： -、+、三种之一加列表内容
    示例：
    (1) - 列表内容
    (2) + 列表内容
    (3) \     列表内容
    注意：- + * 跟内容之间都要有一个空格
  • 有序列表
    语法：数字加小数点(“.”)
    示例：

1.列表内容
2.列表内容
3.列表内容
注意：序号跟内容之间要有空格

  • 列表嵌套
    语法：上一级列表和下一级列表之间敲三个空格即可
    示例：
    + 一级列表
       + 二级列表
       + 二级列表
    效果：
  • 一级列表
    • 二级列表
    • 二级列表

  1. 表格

    语法：
    表头|表头|表头

—|:–:#|— （注：符号#为避免转义加入，使用时不用加）
内容|内容|内容
内容|内容|内容
        第二行纯符号行用来分割表头和内容。-有一个就行，介于内容的长度可能不一，可增加-的数量进行对齐。文字默认居中，-两边加:表示文字居中，-两边加:表示文字居左，-右边加:表示文字居右。
~ 注：原生的语法两边都要用 | 包起来。此处省略 ~
示例：
姓名|技能|排行
–|:–:|–:
刘备|哭|大哥
关羽|打|二哥
张飞|骂|三弟
8. ### 代码

  • 代码引用（>式）
    > hello markdown!

    hello markdown!

>> hello markdown!，”>”的数量为代码缩进的层数

hello markdown!

  • 代码块引用(```)
    >与```组合的示例：

            return 0;

    function fun(){

    echo "这是一句非常牛逼的代码";

    }
    fun();

  1. 注脚
    语法：文本[^1]。
    [^2]: 注脚的解释。
    示例：
    一个具有注脚的文本。^1、^2
  1. LaTeX 数学公式（可能需要平台支持）

    n\in\mathbb N$ 是通过 Euler integral
    $$
    \Gamma(z) = \int_0^\infty t^{z-1}e^{-t}dt,.
    $$
