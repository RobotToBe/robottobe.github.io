hexo —— 新建文章、发布文章

Posted on 2020-01-02 In 分类测试2

新建文章

        可以使用 CSDN 的 Markdown 编辑器编写文章。


        写好之后导出 .md 文件到本地

        然后把它复制到 hexo 博客文件夹的 source 下的 _posts 中。

        复制完之后然后使用文本编辑器（推荐使用：Notepad++）打开它，在开头加上 title，内容自定，如：


        ==注意：黑色方框内 title 的格式共有三行，以 - - - 开始，- - - 结束，title: 后有一个空格。==

发布文章

        接着在 hexo 博客文件夹空白处单击右键，选择 Git Bash Here，

        输入 hexo generate，生成静态文件。

        然后 输入 hexo server，启动本地服务器预览文章，

        若 git 命令行显示本地服务器启动成功，打开浏览器输入 localhost:4000，可以看到写好的文章已经加入到博客里。

        最后输入 hexo deploy，发布文章到网站上。
